Current State of the Industry

One common theme I regularly see with systems is a shift from being isolated to full enterprise integration and from RS-232 to cat5/6. Through systems integration and network technologies control systems are becoming more analogous to traditional IT systems in terms of connectivity. However most systems lack robust security as their unable to run anti-virus, malware or other similar software applications. Additionally, user can now access these systems with mobile devices (phones, tablets, etc.) which often aren't running security applications either. This leads to a situation where none of the communication is private and anyone who has knows the system IP address can gain unlimited access.

The future

More of our home devices will be able to controlled over the internet (Nest Thermostat, August Smart Lock, DVR, ...) but will those connections rival traditional IT security? Right now people are willing to sacrifice security for convenience, only time will tell if that will change.
